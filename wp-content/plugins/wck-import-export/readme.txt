== Description ==

WCK - Import and Export Add-On

This plugin adds a WCK subpage where you can Import and Export settings of WordPress Creation Kit.


== Installation ==

1. Upload wck-import-export folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to WCK => Import and Export subpage and start importing or exporting your options.
